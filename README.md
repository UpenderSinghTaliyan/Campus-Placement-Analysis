# Campus-Placement-Analysis
In my project, I analyzed student placement data to identify the key factors influencing placement outcomes and salary. I began by collecting and cleaning the dataset, where I handled missing values, corrected inconsistencies in CGPA and placement status, and standardized columns using domain knowledge.
Next, I performed exploratory data analysis (EDA) to uncover patterns and trends. The analysis revealed that factors such as internships, projects, and soft skills have a stronger impact on placement than CGPA alone.
To validate these insights, I applied statistical tests like the Z-test and Chi-square test. The results confirmed that both CGPA and internship experience significantly affect placement outcomes.
After the analysis phase, I built machine learning models to make predictions. I used Logistic Regression and K-Nearest Neighbors (KNN) for placement prediction, achieving an accuracy of around 95%. For salary prediction, I implemented Linear Regression, which achieved approximately 78% accuracy.
Additionally, I applied K-Means clustering to segment students into groups and identify hidden patterns within the data. This helped in understanding different student profiles and their placement likelihood.
Finally, I created interactive Power BI dashboards to present insights in a clear and actionable way. These dashboards highlighted key metrics and trends, enabling better decision-making.
Overall, the project demonstrated that a balanced combination of technical skills, internships, projects, and soft skills—along with academics—is essential for placement success.
